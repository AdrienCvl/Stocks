Pour tester le programme :

1 : Sur Android installer Pydroid 3 - IDE for Python 3 et Pydroid reopsitory plugin
2 : Dans pydroid ouvrir le menu; accéder à Pip et installer pandas, numpy, os et openpyxl.
3 : Télécharger le programme depuis Github (dans release télécharger le fichier zip, il contient le programme en .py) et déposer celui-ci sur la mémoire du téléphone et ouvrir celui-ci avc Pydroid (dans l'écran principal logo fichier\Open puis sélectionner le fichier.
3 bis : Il est également possible de copier-coller le le programme directement depuis Github vers Pydroid.
4 : Pour lancer le programme appuyer sur le bouton play en bas à droite.

Pour ouvrir le fichier :

Déplacer l'Excel créé par l'application vers un ordinateur pour l'ouvrir ou sur android installer Excel.
