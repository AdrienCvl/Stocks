Pour tester le programme :

1 : Sur Android installer Pydroid 3 - IDE for Python 3 et Pydroid repository plugin
2 : Dans pydroid ouvrir le menu; accéder à "Pip" et installer : pandas, numpy, os et openpyxl.
3 : Télécharger le programme depuis Github (dans "release" télécharger le fichier zip, il contient le programme en .py) et déposer celui-ci sur la mémoire du téléphone et ouvrir  avec Pydroid (dans l'écran principal logo fichier, cliquer sur "Open" puis sélectionner le fichier.
3 bis : Il est également possible de copier-coller le programme directement depuis Github vers Pydroid.
4 : Pour lancer le programme appuyer sur le bouton play en bas à droite.

Pour ouvrir le fichier :

Déplacer l'Excel créé par l'application vers un ordinateur pour l'ouvrir sur Excel ou sur installer Excel sur Android pour l'ouvrir directement.


A noter : 

Ce programme est un proof of concept, par conséquent tous les produits ne sont pas demandé dans le programme même si le tableau est créé au complet.
De plus lors des vérification le logiciel demande en entrée un caractère correspondant à un produit, ceux pouvant être inscrits sont "o" correspondant à un gilet d'épaule; "x" correspondant à une chaussure de décharge XS et "y" correspondant à une chaussure de décharge S (l'ordre n'est pas celui du tableau car les placards ne sont pas agencés de la même manière, cela permet de gagner du temps dans la salle de stock en évitant les aller-retours dans la salle lors de la prise de stock).
